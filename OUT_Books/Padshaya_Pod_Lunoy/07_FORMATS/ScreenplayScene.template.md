# Screenplay Scene Template

> For feature / episode screenplay output.

```yaml
SCREENPLAY_SCENE:
  scene_id:
  unit_id:
  slugline: INT./EXT. LOCATION - TIME

  scene_contract_ref: SCENE_XXX

  action:
  dialogue:
  subtext_notes:

  act_break: false
  estimated_eighths:

  # Inherited from SCENE_CONTRACT:
  # turn, cost, residue, exit_hook
```

## Format Rules

- Present tense action
- No report voice in action lines
- WorldForce via visible consequence, not narrator thesis
- Subtext over explanation
