# HUD font

Put a Unicode-capable TTF font here:

```text
OUT_RayMicro/data/fonts/hud_unicode.ttf
```

Recommended for the current retro HUD:

```text
GNU Unifont TrueType
```

Why:

```text
Latin
Cyrillic
♥ ❤ ▰ ■ □ ◆ ◇ ✦
wide BMP Unicode coverage
retro pixel-ish look that fits OUT RayMicro
```

Do not commit random font binaries unless the license is checked. Tiny engines do not need surprise legal archaeology.
